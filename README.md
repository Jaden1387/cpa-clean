# cpa-clean

> ⭐ 如果这个项目对你有帮助，欢迎在 GitHub 点个 Star：<https://github.com/xn030523/cpa-clean>

Next.js 全栈美化版，API 与前端都在同一个项目：

- `GET /api/defaults`
- `POST /api/cleanup`

## 一键部署到 Vercel

[![Deploy with Vercel](https://vercel.com/button)](https://vercel.com/new/clone?repository-url=https://github.com/xn030523/cpa-clean)

## 仓库说明

这个仓库是「小呆 - CPA 账户清理工具」的完整源码仓库，用于：

- 提供可直接部署到 Vercel 的在线清理工具
- 维护前端界面、后端清理逻辑和日志展示
- 持续迭代功能与体验（如浏览器本地记住配置）

相关地址：

- 在线地址：`https://clean.daiju.live`
- 代码仓库：<https://github.com/xn030523/cpa-clean.git>
